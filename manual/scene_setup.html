    Scene Setup

    This explains the GameObjects and Prefabs used to get Beatmaps to run properly. There is no need to read through this if you just want to get a beatmap running. The demo scenes are already configured to run properly. This is just to explain the rationale behind how the scene and GameObjects are set up.

    Conceptually, you can group MouseDance into three things:

    1. The Beatmap Runner, which controls the logic
    2. The Hit Objects, which are various different prefabs instantiated onto the Beatmap Canvas
    3. The Beatmap Canvas, which is just a regular Unity UI Canvas, but configured in a way to make it work properly for Beatmaps
