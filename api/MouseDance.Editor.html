Namespace MouseDance.Editor

Editor scripts that MouseDance uses. Only used in the Unity Editor.

Classes

BeatmapRunnerEditor

Provides debug and diagnostic info on top of the Inspector for BeatmapRunner.

HitCircleEditor

Provides buttons to preview the Hit Circle in different Circle Size settings on top of the Inspector for HitCircle.

HitObjectColoursEditor

Provides easier controls for editing colours of a HitObjectColours.

HitObjectSetEditor

Provides copy-paste functionality for HitObjectSet and its sub-classes.

OsuFileImporter

Allows .osu files in a Unity Project to be recognized as TextAsset files.

Util

Minor reused variables for custom Inspectors.
