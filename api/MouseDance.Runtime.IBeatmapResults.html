GetHitAccuracy(Int32)

How close to the Hit Object's time did the player get when they pressed on a Hit Object.

Declaration
float GetHitAccuracy(int hitObjectIdx)
Parameters
Type Name Description
Int32 hitObjectIdx

Index of the Hit Object we are checking.
Returns
Type Description
Single

Normalized value indicating if user pressed Hit Object within correct time window. -1.0 to +1.0 is a hit. Any value outside that (lesser than -1.0 or greater than +1.0) means miss (negative for too early, positive for too late). 0.0 means user pressed exactly at Hit Object time.
